ENVIRONMENT VARIABLES REQUIRED:
================================================================================

At runtime, set these environment variables:

SLACK_BOT_TOKEN
  - Source: Slack app OAuth & Permissions
  - Format: xoxb-<numbers>-<numbers>-<alphanumeric>
  - Scopes: chat:write, app_mentions:read
  - Required: Yes

SLACK_SIGNING_SECRET
  - Source: Slack app Basic Information > App Credentials
  - Format: <alphanumeric>
  - Used for: HMAC-SHA256 signature verification
  - Required: Yes

META_SYSTEM_ACCESS_TOKEN
  - Source: Meta app token (long-lived)
  - Format: <long alphanumeric string>
  - Permissions: ads_read, ads_management
  - Required: Yes

META_AD_ACCOUNT_ID
  - Already exists in .env.local
  - Used for: Fetching ad data

ANTHROPIC_API_KEY
  - Already exists in .env.local
  - Used for: Claude API calls

API ENDPOINTS:
================================================================================

POST /api/slack/events
  Purpose: Slack Events API webhook
  Triggered: app_mention events (@mentions)
  Request: Slack sends JSON with event data
  Response: 200 OK (immediate), processing async
  Verifies: X-Slack-Signature header

POST /api/slack/interactions
  Purpose: Slack Interactive Components webhook
  Triggered: Button clicks on messages
  Request: Slack sends form-encoded payload
  Response: 200 OK (immediate), processing async
  Verifies: X-Slack-Signature header

GET /api/slack/status
  Purpose: Configuration status check
  Triggered: Settings page load
  Request: None
  Response: JSON { configured, hasBotToken, hasSigningSecret }
  Verifies: None
